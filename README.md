# Tutti-Frutti Helper
Cansado de que mi novia me gane en el tutti frutti hice un bot que le hiciera frente (Casi que no le gana aveces)

## Requirements

```
pip install tabulate
pip install beautifulsoup4
```
## Usage
```
py 'Tutti Frutti.py'
```
